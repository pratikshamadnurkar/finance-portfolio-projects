# Portfolio Optimization – Efficient Frontier

## Overview

This project applies Modern Portfolio Theory (MPT) to identify the optimal allocation of assets based on expected return and investment risk. Using historical market data, the model constructs an Efficient Frontier and identifies portfolios that maximize risk-adjusted returns.

The project reflects portfolio management techniques commonly used by asset managers, investment analysts, and wealth management professionals.

---

## Business Problem

Investors seek to maximize returns while minimizing unnecessary risk.

Without quantitative portfolio optimization, investment decisions may overlook diversification benefits and fail to achieve an appropriate balance between risk and return.

---

## Objective

Develop an investment optimization model capable of:

- Constructing the Efficient Frontier
- Optimizing asset allocation
- Maximizing risk-adjusted returns
- Supporting investment decision-making

---

## Dataset

The model uses historical market price data for multiple financial assets.

The dataset includes:

- Asset Returns
- Risk Measures
- Correlation Matrix
- Covariance Matrix

---

## Methodology

### Data Preparation

- Return Calculation
- Correlation Analysis
- Covariance Matrix Construction

### Portfolio Optimization

Calculated:

- Portfolio Return
- Portfolio Risk
- Sharpe Ratio

Generated:

- Efficient Frontier
- Minimum Variance Portfolio
- Maximum Sharpe Portfolio

---

## Results

The model identifies optimal portfolio allocations across different levels of investment risk and demonstrates the benefits of diversification using Modern Portfolio Theory.
