# GDP Forecasting – ARIMA Model

## Overview

This project demonstrates the application of ARIMA time-series models to forecast GDP trends using Eurostat macroeconomic data. The model provides forecasts that can support economic analysis and long-term planning.

The project reflects forecasting techniques commonly used by economists, central banks, and corporate strategy teams when evaluating future economic conditions.

---

## Business Problem

Macroeconomic conditions directly influence business performance, investment decisions, and financial planning.

Organizations require reliable economic forecasts to anticipate changes in demand, investment opportunities, and financial risk.

---

## Objective

Develop an ARIMA forecasting model capable of:

- Forecasting GDP trends
- Identifying economic patterns
- Supporting macroeconomic analysis
- Evaluating future growth scenarios

---

## Dataset

Source:

Eurostat

The dataset contains historical GDP observations across multiple reporting periods.

---

## Methodology

### Data Preparation

- Data Cleaning
- Time-Series Construction

### Time-Series Analysis

Performed:

- Stationarity Testing
- Differencing
- Model Selection

### Forecasting

Generated GDP forecasts using ARIMA models and compared forecast values against historical observations.

---

## Results

The model successfully captures historical GDP trends and demonstrates the practical application of econometric forecasting techniques for economic analysis.

---

## Business Value

Macroeconomic forecasts support organizations by:

- Improving strategic planning
- Supporting budgeting and forecasting
- Assessing economic risk
- Informing long-term investment decisions
