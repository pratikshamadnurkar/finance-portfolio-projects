# FP&A Dashboard – Ferrari Financial Performance

## Overview

This project demonstrates how an FP&A team can monitor financial performance through interactive KPI reporting and management dashboards.

The dashboard analyzes Ferrari's financial performance across multiple reporting periods and provides visibility into revenue growth, profitability trends, and capital structure metrics.

---

## Business Problem

Finance teams need timely access to financial information to support planning, budgeting, forecasting, and strategic decision-making.

Traditional spreadsheet reporting can make it difficult to identify performance trends and communicate insights effectively to stakeholders.

---

## Objective

Develop an interactive FP&A dashboard that enables:

- KPI monitoring
- Financial performance analysis
- Trend identification
- Management reporting
- Strategic decision support

---

## Dataset

Source: Ferrari Annual Reports

Financial metrics include:

- Revenue
- EBITDA
- Net Income
- Total Assets
- Total Debt
- Shareholder Equity

---

## Methodology

### Data Preparation

- Collected financial statement data
- Cleaned and structured reporting metrics
- Standardized financial calculations

### KPI Development

Calculated:

- Revenue Growth
- EBITDA Margin
- Net Profit Margin
- Debt-to-Equity Ratio
- Return on Assets

### Dashboard Development

Built an interactive dashboard using:

- R
- Shiny
- Plotly

---

## Dashboard Features

### Revenue Analysis

Track revenue performance across reporting periods.

### Profitability Analysis

Monitor margins and profitability trends.

### Capital Structure Monitoring

Evaluate leverage and financing decisions.

### Interactive Visualizations

Enable drill-down analysis for management reporting.

---

## Results

The dashboard provides a centralized view of Ferrari's financial performance and enables users to identify trends, evaluate operational performance, and support financial planning activities.

---

## Business Value

This project mirrors the reporting environment commonly used by FP&A teams to:

- Support budgeting and forecasting
- Monitor key performance indicators
- Improve management reporting
- Facilitate strategic decision-making
